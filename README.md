### Usage

```python
import Logger

# [Your's imports]

# Initialize logger 
LOG = Logger("Module Name")

# Use it!
LOG.core("TEXT")
LOG.info("TEXT")
LOG.warn("TEXT")
LOG.error("TEXT")
```
